# L.L Beauty Website MVP

This repository currently contains a polished static homepage prototype for the `L.L Beauty` brand. It is designed as a premium landing page that combines luxury product discovery, salon positioning, and franchise lead generation in one flow.

## What is implemented now

- Premium mobile-first homepage in `index.html`
- White, gold, and soft beige luxury visual system in `style.css`
- Header and search
- Hero with product and franchise CTAs
- Product showcase cards
- Salon / Lounge / Kiosk experience section
- Franchise models and lead form UI
- Commerce and CRM flow overview
- Footer with brand and contact links

## Recommended production stack

- Frontend: Next.js + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- Authentication: Firebase Auth or OTP provider integration
- Payments: Razorpay first, Stripe later
- Hosting: Vercel for frontend, Render or AWS for backend

## Suggested next build steps

1. Convert the current static layout into a Next.js app structure.
2. Break the homepage into reusable components like `Header`, `Hero`, `ProductCard`, and `FranchiseForm`.
3. Add product and franchise APIs with Express and MongoDB models.
4. Integrate OTP verification for login and franchise lead validation.
5. Add cart, checkout, Razorpay payment flow, and order persistence.
6. Add CRM dashboard and admin views for lead and order management.
